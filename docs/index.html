PowerID Privacy Policy

Last updated: 2026-07-06

PowerID is a Windows battery monitoring app. This policy explains what data PowerID does and does not access, in plain terms.

What PowerID accesses

PowerID reads battery and power status information directly from your device's ACPI/Windows battery APIs — charge level, voltage, wattage, temperature, cycle count, and charging state. This data is read locally and used only to display it in the app's own interface.

What PowerID does not do

  • PowerID makes no network requests. It has no internet access code path at all.
  • PowerID does not collect, transmit, sell, or share any data with anyone, including the developer.
  • PowerID has no telemetry, analytics, or crash reporting.
  • PowerID does not require or use a user account, sign-in, or any personal information.
  • PowerID does not use cookies or any tracking technology (it isn't a web app).

Data storage

Any settings PowerID saves (such as window preferences) are stored locally on your device only, using standard Windows application data storage. Nothing leaves your device.

Changes to this policy

If PowerID's data practices ever change, this page will be updated accordingly before that version is published.

Contact

Questions about this policy can be filed as an issue at github.com/ogden-marrow/PowerID_For_Windows/issues.
